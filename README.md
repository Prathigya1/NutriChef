# 🍽️ NutriChef

NutriChef is an Android application developed using Java in Android Studio. The app helps users discover healthy recipes and provides nutritional information for various food items.

## 📱 Features

- 🥗 Browse healthy recipes
- 🔍 Search recipes
- 🍲 View recipe details
- 📊 Nutrition Calculator
- 🎨 User-friendly interface

## 🛠️ Technologies Used

- Java
- Android Studio
- XML
- Gradle

## 📂 Project Structure

```
NutriChef/
├── app/
├── gradle/
├── build.gradle
├── settings.gradle
├── gradle.properties
└── README.md
```

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/Prathigya1/NutriChef.git
```

2. Open the project in Android Studio.

3. Sync Gradle.

4. Run the app on an Android emulator or physical device.
